# Serato-Library-Parser
Serato Library Parser è uno strumento leggero e portabile che permette di leggere il file database V2 di Serato DJ e convertirlo in formati aperti come JSON e XML. È completamente read-only, non modifica i file originali e offre un modo semplice e veloce per esportare, analizzare o migrare la propria libreria musicale.
